조건문과 반복문
- 조건문 : 조건을 만족할 때만 시행
- 반복문 : 조건을 만족하는 동안 시행

if문
- 조건식이 참일 때 괄호 안의 문장들을 수행
- if(조건식) { }

블럭 { }
- 여러 문장을 하나로 묶어주는 것(한문장은 생략 가능)

if-else문
- 조건식이 참일 때와 거짓일 때로 나눠서 처리(둘 중의 하나)
- if(조건식) { 
  } else { }
  
if-else if문
- 여러 개의 조건식을 포함한 조건식(여러 개 중의 하나)
- if(조건식1) {
  } else if(조건식2){
  } else if(조건식3){
  } else { }

중첩 if문 - if문 안의 if
- if(조건식1) {
	if(조건식2) { 
	 } else { }
  } else { }
