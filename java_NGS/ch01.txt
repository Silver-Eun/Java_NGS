Java
- 프로그래밍 언어
- 컴퓨터 프로그램(어플리케이션)을 만드는데 사용
- 실행환경(JRE) + 개발도구(JDK) + 라이브러리(API)
- 쉽고 빠르게 어플리케이션 개발 가능

어디에 쓰이나
- PC 어플리케이션(IntelliJ, Eclipse)
- 웹 어플리케이션(SPringBoot)
- 모바일 어플리케이션(Android Studio)
- 빅데이터(hadoop)
- 게임, 과학, 소형기기 등

왜 자바인가
- 다양한 분야에서 활발히 사용
- 20년 동안 프로그래밍 언어 1 ~ 2위
- 배우기 쉽고 풍부한 학습자료
- 모던 프로그래밍 언어(객체지향 + 함수형)
- 취업에 유리(실무에서 제일 많이 사용)

자바의 역사
- JDK(자바 개발 도구)
- J2SE 1.2(Java2)
- J2SE 5.0
- J2SE 8

자바의 특징
- 배우기 쉬운 객체지향 언어(프로그래밍 언어 + 객체지향 개념)
- 자동 메모리 관리(가비지 컬렉터 GC)
- 멀티 쓰레드 지원
- 풍부한 라이브러리로 쉽게 개발 가능
- 운영체제에 독립적

자바 가상 머신(JVM)
- 자바 프로그래밍이 실행되는 가상 컴퓨터(VM)
- 한번 작성하면 어디서든 실행

자바 API
- Java 프로그램을 만드는데 필요한 주요 기능을 미리 만들어서 제공

자바 API 문서
- Java API가 제공하는 기능에 대한 상세정보 제공

javax.exe
- 자바 컴파일러, 사람이 작성한 문장을 기계어로 번역
- 소스 파일(*.java)을 클래스 파일(*.class)로 변환

java.exe
- 자바 인터프리터, 자바프로그램(클래스파일)을 실행

클래스
- 자바 프로그램의 단위, 자바 프로그램은 클래스들로 구성

main 메서드
- 자바프로그램의 시작점, 이 메서드 없이 실행불가
